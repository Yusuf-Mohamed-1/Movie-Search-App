# Movie Search App

A movie search application built with HTML, CSS, and JavaScript using the TMDB API.

## Features

- Search for movies by title
- Detailed movie information popup
- Movie poster display
- Movie overview
- Release date
- Movie rating
- Total vote count
- Runtime information
- Original movie language
- Production country
- Budget and revenue details
- Watch movie trailers
- Cast section with actor photos
- Director section with director photos
- Loading state
- Error handling
- No results state
- Responsive design

## Technologies Used

- HTML
- CSS
- JavaScript
- TMDB API
