# 🎬 Movie Search App

A modern movie search application built using HTML, CSS, and JavaScript with the TMDB API.  
This project demonstrates working with APIs, dynamic UI rendering, and handling complex movie data in a real-world project.

---

## 🔗 Live Demo
https://yusuf-mohamed-1.github.io/Movie-Search-App/

---

## ✨ Features

- 🔍 Search for movies by title
- 🎬 Detailed movie information popup
- 🖼️ Movie poster display
- 📝 Movie overview
- 📅 Release date
- ⭐ Movie rating and vote count
- 👥 Number of votes
- ⏱️ Runtime information
- 🌍 Original language & production country
- 💰 Budget and revenue details
- 🎥 Watch movie trailers
- 🎭 Cast section with actor images (6 actors)
- 🎬 Directors section with images (3 directors)
- ⚡ Loading state handling
- ❌ Error handling
- 🔎 No results state
- 📱 Fully responsive design

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- TMDB API
- Fetch API
